# 개요

이 공간은 개인적인 학습, 실습, 적용기, 디버깅 기록을 정리하기 위한 공간입니다.  
특정 기술에 제한되지 않고, 실제 개발 과정에서 마주치는 다양한 문제와 해결 과정을 함께 기록합니다.

> 일부 문서의 경우 손으로 정리한 내용에서 맞춤법이나 구조를 생성형 AI를 이용하여 정리하였습니다.  
> **하지만 콘텐츠는 작성자가 직접 신뢰도 있는 수업이나 공식문서를 학습하고 정리 한 내용을 기반으로 합니다.**

---

## 1. CS노트

정규 수업 또는 온라인 강의를 들으며 정리한 **이론 중심 노트**입니다.  
주요 목적은 **이해한 내용을 자신의 언어로 다시 정리하는 것**입니다.

---

## 2. 튜토리얼 / 실습 (Tutorial & Practice)

새로운 기술을 배우기 위해 진행한 실습 기반 정리입니다.
코드 나열보다는 **동작 흐름**과 **핵심 개념**을 중심으로 정리합니다.

---

## 3. 적용기 (Case Study)

프로젝트나 개인 토이 프로젝트에서 실제로 기술을 적용하면서 얻은 경험을 기록합니다.

문서화 포인트는 "**왜 이런 결정을 했는가?**"입니다.

---

## 4. 디버깅 기록 (Debugging Log)

개발 중 발생한 문제를 분석하고 해결한 과정을 남기는 공간입니다.

문제 상황 → 분석 과정 → 해결 방법 → 재발 방지 포인트  
형식으로 정리합니다.
